# VF视频播放SDK
> H5页面视频播放器，有问题请

视频SDK致力于提供属于公司自身的播放SDK，使得使用者通过简单的流程接入即可使用，目前SDK本身支持mp4，m3u8等主流视频格式。

## 安装

npm方式:

```sh
npm install vf-mediaPlayer
```

或者选择直接引入:

```sh
<script src='yourPath/VFMediaPlayer.js'></script>
```
